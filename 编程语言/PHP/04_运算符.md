## 一、介绍

```php
# 概览
	+	相加
	-	相减
	*	相乘
	/	相除
	%	取余
	++	加加
	--	减减
	.	连接、用在字符串
```



## 二、赋值运算符

```
	=	赋值运算符
	+=	先加，后赋值
	-=	先减，后赋值
	*=	先乘，后赋值
	/=	先除，后赋值
	%=	先取余，后赋值
	.=	先连接，后赋值
```



## 三、比较运算符

```
	>	大于
	>=	大于等于
	<	小于
	<=	小于等于
	==	等于
	!=	不等于
	===	恒等于
	!==	恒不等
```



## 四、逻辑运算符

```pph
	and和&&	与：两个真，返回真。有一个是假，返回
	or和||	或：一个真，返回真。两个真，返回真。两个假，返回假
	xor		 异或：一个真，返回真。两个真，返回加，两个假，返回假
	!		 非：一个真，返回真。两个真，返回假，两个假，返回假
```
